# EC2 Instance Types

Instance types define:
- CPU
- Memory
- Storage
- Network capacity

---

## Categories

### General Purpose
Balanced CPU & memory  
Example: t3, t4g

### Compute Optimized
High CPU performance  
Example: c5

### Memory Optimized
High RAM  
Example: r5

### Storage Optimized
High disk throughput  
Example: i3

---

Choosing correct instance type improves performance and cost efficiency.
