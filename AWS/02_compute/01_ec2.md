# Amazon EC2 (Elastic Compute Cloud)

## What is EC2

Amazon EC2 provides resizable virtual servers in the cloud.

It allows users to launch and manage virtual machines called instances.

---

## Key Features

- On-demand virtual servers
- Multiple instance types
- Auto Scaling support
- Elastic IP support
- Integration with VPC
- Security Groups for firewall control

---

## EC2 Instance Lifecycle

1. Launch
2. Running
3. Stopped
4. Terminated

Stopped instances do not incur compute charges.
Terminated instances cannot be recovered.

---

## Pricing Models

- On-Demand
- Reserved Instances
- Spot Instances
- Savings Plans

---

## Important Concepts

AMI (Amazon Machine Image) → Template to launch EC2  
Security Group → Virtual firewall  
Key Pair → SSH authentication
