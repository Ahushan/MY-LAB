# Amazon Machine Image (AMI)

## What is AMI

An AMI is a pre-configured template used to launch EC2 instances.

It contains:
- Operating system
- Application server
- Applications
- Configuration settings

---

## Types of AMI

- Public AMI
- Private AMI
- AWS Marketplace AMI

---

## Important Point

AMI is region-specific.

To use in another region, it must be copied.
