# Cloud Models

## What is Cloud

Cloud computing is the delivery of computing resources such as servers, storage, networking, databases, and software over the internet instead of owning physical hardware.

Cloud follows:
- On-demand access
- Pay-as-you-go pricing
- Resource pooling
- Rapid elasticity
- Measured service

---

## Characteristics of Cloud Computing

1. On-demand self-service
2. Broad network access
3. Resource pooling
4. Rapid elasticity (auto scaling)
5. Measured service (usage-based billing)

---

## Cloud Service Models

### IaaS (Infrastructure as a Service)

Provides virtualized infrastructure.

Examples:
- Amazon EC2
- Amazon EBS
- VPC

User manages:
- OS
- Applications
- Runtime
- Data

Provider manages:
- Physical servers
- Storage hardware
- Networking hardware

---

### PaaS (Platform as a Service)

Provides managed application platform.

Examples:
- AWS Elastic Beanstalk
- AWS Lambda

User manages:
- Application code
- Data

Provider manages:
- OS
- Runtime
- Infrastructure

---

### SaaS (Software as a Service)

Provides fully managed software over the internet.

Examples:
- Gmail
- Salesforce
- Microsoft 365

User manages:
- Only usage & configuration
